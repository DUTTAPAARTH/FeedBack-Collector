📥 HOW TO USE:
==============

OPTION 1: Extract Archive
────────────────────────
$ tar -xzf feedback-collector.tar.gz
$ cd feedback-collector

OPTION 2: Use Folder Directly
──────────────────────────────
$ cd feedback-collector

SETUP:
──────
$ npm install
$ cd client && npm install && cd ..

RUN BACKEND:
────────────
$ npm start
# Runs on http://localhost:5000

RUN FRONTEND:
─────────────
$ cd client && npm start
# Opens http://localhost:3000
